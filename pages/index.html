OpenTubeX Flatpak

Flatpak packages for OpenTubeX.

Install

Add the remote once:

flatpak remote-add --user --if-not-exists opentubex \
  https://flatpak.opentubex.org/opentubex.flatpakrepo

flatpak install --user opentubex org.opentubex.OpenTubeX

Update

flatpak update --user org.opentubex.OpenTubeX

Offline bundles

You can also download .flatpak bundle files from GitHub Releases and run flatpak install --user ./org.opentubex.OpenTubeX-*.flatpak. Bundle installs are not tied to this remote, so they do not receive flatpak update or updates through your software store (for example Discover on KDE).
