OpenTubeX Flatpak

Flatpak packages for OpenTubeX.

Install

Add the remote once:

flatpak remote-add --if-not-exists opentubex \
  https://flatpak.opentubex.org/opentubex.flatpakrepo

flatpak install opentubex org.opentubex.OpenTubeX

Nightly builds

Install the opt-in development branch from the same remote:

flatpak install opentubex org.opentubex.OpenTubeX//nightly

Update

flatpak update org.opentubex.OpenTubeX

Offline bundles

You can also download .flatpak bundle files from GitHub Releases and run flatpak install ./org.opentubex.OpenTubeX-*.flatpak. Bundle installs are not tied to this remote, so they do not receive flatpak update or updates through your software store (for example Discover on KDE).
